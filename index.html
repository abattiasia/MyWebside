AI Models

Learn about different AI models such as OpenAI and Ollama used to build chat applications.


# Example of using OpenAI with Python
import openai

openai.api_key = 'your-api-key'

response = openai.Completion.create(
    engine="text-davinci-003",
    prompt="Explain generative AI.",
    max_tokens=100
)

print(response.choices[0].text.strip())

# Example of using Ollama with Python
import ollama

response = ollama.generate(
    model="llama-2",
    prompt="What is generative AI?"
)

print(response)
